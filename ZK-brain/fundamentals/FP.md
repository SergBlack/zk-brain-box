# FP
In FP we operate with functions, mostly in a declarative style.
We build apps by composing pure functions, avoiding share state, mutable data and side-effects.

Ideas:
-   Pure functions
-   Function composition
-   Avoid shared state
-   Avoid mutating state
-   Avoid side effects
